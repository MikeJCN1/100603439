How to use the program:

1. Run the main file
2. Input the test file e.g. "test1.txt" , there are 5 test files included.
3. The results of the test will be displayed and the main file will run the run_manual function.
4. This function allows you to type expressions in a single or multi line format.
5. Press enter on a blank line to receive the results of an expression.
6. Type "quit" when you want to exit the loop.
